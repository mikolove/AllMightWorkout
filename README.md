# AllMightWorkout

Android Workout personal application learning Clean Architecture, Modularization and testing new Android libs.

App feature content :

  - Clean architecture paired with Model View Intent pattern
  - Module
  - Version catalog
  - Android Jetpack component( Compose, ...)
  - Cache : room 
  - Network : firestore
  - Kotlin
  - Di : hilt-dagger
  - Unit Test
  - Custom gradle setup
